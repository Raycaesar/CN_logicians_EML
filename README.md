<h2 id="chinese">中文简介</h2>

本项目致力于重新审视20世纪初中国逻辑学家在现代模态逻辑发展初期所做出的先驱性贡献。本仓库托管了该项目的官方展示网站源代码。

### 🌟 核心研究版块
1. **沈有乾与 Ladd-Franklin**：探寻代数逻辑到现代模态逻辑的演变。
2. **沈有鼎与 S4/S5**：哈佛时期的修正工作与模态语义学贡献。
3. **汤藻真与 S2**：早于塔斯基的拓扑/代数语义学先声。
4. **金岳霖**：早期模态逻辑在中国的传播枢纽。

<h2 id="english">English Introduction</h2>

This project is dedicated to re-examining the pioneering contributions of early 20th-century Chinese logicians to the development of modern modal logic. This repository hosts the source code for the project's official website.

### 🌟 Core Research Themes
1. **Eugene Shen & Ladd-Franklin**: Tracing the evolution from algebraic logic to modern modal logic.
2. **Shen Yuting & S4/S5**: His corrections at Harvard and contributions to modal semantics.
3. **Tang Tsao-Chen & S2**: Pre-Tarskian topological/algebraic semantics.
4. **Jin Yuelin**: The academic hub for the early transmission of modal logic in China.

### 🛠️ Tech Stack & Deployment
The website is built with pure front-end technologies (HTML5, CSS3, Vanilla JavaScript) without complex dependencies, ensuring long-term accessibility and easy maintenance of academic resources.
* Hosted on [GitHub Pages](https://raycaesar.github.io/CN_logicians_EML/).
* Features a lightweight, front-end bilingual (EN/ZH) switching mechanism.
